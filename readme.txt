                                            SpyChat
SpyChat is a  terminal chat app using python that allows you to do many things like-
 ->Updating status
 ->Add a friend 
 ->Send and  Receive secret messages hidden safely inside an image
 ->Displaying chat history.
 The chat app will not be using a database or any other form of persistent data storage and will rely on the computer's non-persistent memory (RAM)
